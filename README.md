# Genshin-Glyphs

🌏 **[English](README_en.md)** | 简体中文

## 简介

这个仓库储存了游戏《原神》中的架空文字的字体。

## 目前游戏里有哪些文字？

游戏里所有的文字都包含在下面的列表里面。如果前面打了个勾，
表示这个文字的字体已制作完成，可以在这个仓库里找到。

- [ ] 通用文字（蒙德）
- [ ] ~~璃月文字~~
- [X] 稻妻文字
- [ ] ~~石根文字（稻妻）~~
- [X] 坎瑞亚文字

## 渲染出错？

如果这里的字体在您的软件中出现太大、太小或渲染出错的情况，请在上方提出 Issue，谢谢。

## 已知问题

### 稻妻文字

部分字母来源于游戏内的模糊截图，显示效果可能与实际不符，这些字母是：
```
A B C D F J T W Y Z
```

部分字母还没有被制作成字体，这些字母是：
```
L P Q V X
```
